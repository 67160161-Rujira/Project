# 🏠 แอปพลิเคชันทำนายราคาบ้าน (House Price Prediction)

## 📌 ภาพรวมของโปรเจค
โปรเจคนี้เป็นเว็บแอปพลิเคชันที่พัฒนาด้วย Streamlit เพื่อใช้ในการทำนายราคาบ้าน โดยอาศัยเทคนิค Machine Learning  
ผู้ใช้สามารถกรอกข้อมูลเกี่ยวกับอสังหาริมทรัพย์ เช่น จำนวนห้อง ขนาดพื้นที่ ระยะทางจากเมือง ฯลฯ  
และระบบจะคำนวณราคาบ้านโดยประมาณ พร้อมแสดงระดับความมั่นใจ (Confidence)

---

## 🎯 วัตถุประสงค์
- เพื่อศึกษาการนำ Machine Learning มาประยุกต์ใช้กับข้อมูลจริง
- เพื่อสร้างระบบที่ผู้ใช้ทั่วไปสามารถใช้งานได้ง่าย
- เพื่อพัฒนา Web Application สำหรับการวิเคราะห์ข้อมูล

---

## 🚀 ความสามารถของระบบ
- 🔮 ทำนายราคาบ้าน (แสดงผลทั้ง AUD และ THB)
- 📊 แสดงค่าความมั่นใจของโมเดล (Confidence Score)
- ✅ มีระบบตรวจสอบข้อมูล (Input Validation)
- 💡 มีคำอธิบายแต่ละตัวแปร (Feature Explanation)
- 🎨 อินเทอร์เฟซใช้งานง่ายและสวยงาม

---

## 🧠 โมเดลที่ใช้
- อัลกอริทึม: Random Forest Regressor
- พัฒนาโดยใช้ Scikit-learn
- มีการทำ Data Preprocessing เช่น:
  - Encoding ข้อมูลประเภทข้อความ (Categorical Data)
  - Scaling ข้อมูลตัวเลข (Numerical Data)

---

## 📥 วิธีการใช้งาน
1. กรอกข้อมูลอสังหาริมทรัพย์ เช่น:
   - จำนวนห้อง (Rooms)
   - จำนวนห้องน้ำ (Bathrooms)
   - ขนาดที่ดิน (Land Size)
   - ระยะห่างจากตัวเมือง (Distance)
2. เลือกภูมิภาค (Region)
3. กดปุ่ม **Predict Price**
4. ระบบจะแสดง:
   - ราคาที่คาดการณ์ (AUD และ THB)
   - ค่าความมั่นใจ (Confidence)
